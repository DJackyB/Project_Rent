# 卡牌效果配置建议 v1（Excel 为唯一数据源）

> 说明：以下为可直接填入 `instantEffect/settleEffect` 等字段的建议值。
> 支持使用 `|` 组合多个效果。

## 事件卡

| 卡名 | 建议配置 |
|---|---|
| 强制退租 | `EvictTenantInSelectedRoom` |
| 短期续租 | `AddTenantDurabilityInSelectedRoom;2` |
| 临时加租 | `AddMoney;100` |
| 永久加租 | `AddTenantDurabilityInSelectedRoom;20` |
| 混响视听 | `TriggerSelectedRoomSettle` |
| 强制搬迁 | `MoveTenantToEmptyRoom` |
| 全网发布 | `SpawnRandomTenantInSelectedRoom` |

## 合同卡

| 卡名 | 建议配置 |
|---|---|
| 社恐 | `AddMoneyByEmptyRooms;50`（settleEffect） |
| 环境租 | `AddMoneyByRoomCount;100`（settleEffect） |

## 备注

- 以上“需要选房间”的效果都包含 `SelectedRoom` 关键字，出牌时会进入选房间流程。
- 导表后生效，不需要手动编辑 ScriptableObject。
